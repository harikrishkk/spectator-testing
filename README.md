# Spectator testing 

* create a fresh angular app .
* Add jest instead of karma / jasmine - sudo ng add @briebug/jest-schematic
* ng test --watch to continuously watch changes
* ng test --collect-coverage. to get test coverage
* spectator and ngMocks used with jest 
* https://github.com/thymikee/jest-preset-angular
